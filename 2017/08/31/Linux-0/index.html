解决ubuntu使用的一个小问题

RunLove
2017-08-31

使用的是下基于Ubuntu的Linux Mint 18(Sarah)x64系统, 因为一开始的时候对于Linux的使用不大熟练, 安装完之后选择了中文语言, 也因为上手简单, 很多东西都是定制好的,
比如安装完成Home目录下就已经为你创建了包括音乐, 下载, 模板, 桌面等空目录, 问题就在这里, 因为不喜欢中文文件名, 所以就手动把桌面这个文件名改成了英文desktop,
本来没什么事情的, 继续折腾东西, 等我下次重启之后, 发现桌面上多了很多东西(因为我是不喜欢放很多东西在桌面的, 这一点相信有很多人跟我一样, 简洁即美)，那么只放了一张图片在桌面上的我的系统桌面怎么突然多了那么多东西呢? 定睛一看, 原来都是我Home目录下的东西, 怎么都跑到桌面上了…我还试了一下打开文件管理器看看能不能把它们移动回去, 好蠢:)…好吧, 既然知道出了什么问题了, 那就对症下药好了.
